##Условие задачи
Исходное уравнение:
```
u_xx + 4u_xy - 5u_yy - 3u_x + 3u_y = 3y + 3x
```
Начальные условия:
```
  u|(y-2x=0) = 2 + 6x^2 + 4x 
u_y|(y-2x=0) = 5x + 2
```
